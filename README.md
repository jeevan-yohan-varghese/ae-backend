# AE TASK Backend

Backend for events app


## Features

- [x] User authentication
- [x] CRUD Event for admin
- [x] Register For Event
- [x] Checkin for Event
- [x] Create Venue
- [x] Delete Venue
- [x] Get ALl Events
- [x] Get My Events
## Features

- Postgresql with sequelize ORM

- JWT based authentication

- middlewares (apikey, auth)

- Firebase auth email/google
